# Проект: Асинхронная загрузка персонажей Star Wars

## Описание

Проект предназначен для асинхронной загрузки данных о персонажах вселенной Star Wars с публичного API (https://www.swapi.tech/) и сохранения их в базе данных PostgreSQL.

## Функциональность

- Асинхронные HTTP-запросы к API с использованием aiohttp
- Асинхронная работа с базой данных через SQLAlchemy и asyncpg
- Загрузка данных о персонажах, включая:
  - Основные атрибуты (имя, год рождения, цвет глаз и т.д.)
  - Название родной планеты (через дополнительный запрос)
  - Списки фильмов, видов, кораблей и транспорта в виде строк с названиями через запятую
- Пакетная обработка данных с разбивкой на чанки

## Структура проекта

- `async_requests.py` — основной скрипт для загрузки данных
- `models.py` — модели SQLAlchemy
- `migrate.py` — скрипт для создания таблиц в базе данных
- `requirements.txt` — зависимости
- `docker-compose.yml` — конфигурация для запуска PostgreSQL в Docker
